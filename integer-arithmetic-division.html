An optional task: Integer Division

Division of integers is, of course, successive subtraction. So 13 divided by 4 amounts to noting how many time 4 can be removed from 13 and keeping track of the remainder. Keep this in mind as you write the recursive method, divTwo, in the class Divide to do this. The divTwo signature is written for you. Do not change it!

Additional complexity exists in that there are two things to report, the quotient and a reminder. The right way to handle this is to create a new class with two fields. Create the class DivVals with exactly two fields, quotient and remainder. Required methods include a constructor that sets both fields to 0 on instantiation plus getter and setter methods for each field.

This task is interesting, in part, because it exercises more of the object oriented (OO) elements of Java. It illustrates simple OO design and the fact that all the details must be addressed and correctly implemented.
