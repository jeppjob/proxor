Binary Trees: Implementing other operations on ordered Binary Trees

Now that you understand a bit more about Binary Trees, do the following programming tasks to dive deeper.

Programming Task: Print Tree

Instructions:

Step 1: Using the Eclipse IDE open the Java Perspective. Go to the Package Explorer and open recursion/src/binarytrees/ Tree.java.

Step 2: In the outline view click on printTree(Tree):void. The central window will scroll to the method printTree and scroll to it.

Step 3: Write the recursive routine that prints the tree from left to right.

Step 4: Watch printTree.mov to see how things work when you have correctly coded the routine.

Programming Task: Find Node

Instructions:

Step 1: Using the Eclipse IDE open the Java Perspective. Go to the Package Explorer and open recursion/src/binarytrees/ Tree.java.

Step 2: In the outline view click on findNode(Tree, String):Tree. The central window will scroll to the method findNode and scroll to it.

Step 3: Write the recursive routine returns the first node with data that matches key. If there is no such node it returns null.

Step 4: Watch find.mov to see how things work when you have correctly coded the routine.
