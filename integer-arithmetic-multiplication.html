Programming Task: Integer Multiplication

Step 1: From recursion>src>intarithmetic open Multiply.java

Step 2: Multiplication of integers is, of course, successive addition. So 3 times 4 is the exact same thing as 4 + 4 + 4. Keep this in mind as you write the recursive method, multTwo, in the class Multiply to do this. The method signature for multTwo is written for you. Do not change it! Think about the order in which operations need to take place to perform successive additions.

Step 3: Use both addTwo and subTwo for full credit. This means don’t use built-in binary arithmetical operators {+,-,*,/} in the class multiply.

If you're finding Multiplication difficult:

If you you computed the product but added a third parameter that is not OK. Try this instead:

• Stick with on our example of 3 * 4. Convert the multiplication to an addition to the answer to a more simple multiplication. 4 + (2 * 4)

• Keep going with the process until you hit the bottom, the base case. 4 + 4 + (1 * 4)

• Generalize this and you're done!
